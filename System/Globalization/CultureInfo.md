##### System.Globalization.CultureInfo
## CultureInfo

Get All Cultures
``` csharp
CultureInfo.GetCultures(CultureTypes.SpecificCultures)
```

Get the current culture
``` csharp
CultureInfo.CurrentCulture
```

Set a new culture
``` csharp
CultureInfo.DefaultThreadCurrentCulture = new CultureInfo("jp-JN")
```
